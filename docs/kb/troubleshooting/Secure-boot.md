# Secure boot

If you have troubles getting iPXE to initialize you may need to disable Secure boot.

Depending on how you Install FOG Server you may need to disable Secure boot for the host machine. Disabling Secure boot may also be required to deploy/capture/register as well. 

The FOG Project at this time does not have the resources necessary to provide a Secure-Boot ready build.
